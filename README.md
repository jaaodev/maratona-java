### ☕ Maratona Java - DevDojo
Repositório dedicado aos estudos realizados durante o curso Maratona Java, ministrado pelo William Suane no canal DevDojo. Aqui armazeno todos os códigos, exercícios e anotações sobre o curso.

### 🚀 Tecnologias e Conceitos
Linguagem: Java 17+

Paradigma: Orientação a Objetos (POO)

Temas abordados: Métodos, herança, polimorfismo, exceções, coleções, concorrência e mais.

### 📂 Estrutura do Repositório
O projeto está organizado por pacotes para facilitar a localização:

* **javacore/src/**: Onde a mágica da POO acontece.
    * `AsobrecargaMetodos`
    * `Bconstrutores`
    * `CblocoInicializacao`
    * `DmodificadorEstatico`
    * `Eassociacao`
    * `Fsobrescrita`
    * `GmodificadorFinal`
    * `Henumeracao`
    * `IclassesAbstratas`
    * `Jinterfaces`
    * `Kpolimorfismo`
    * `Lexcecoes`
    * `modificadorAcesso`

### 🛠️ Como executar
Para rodar qualquer um dos exemplos localmente, você precisará do JDK 17 (ou superior) instalado.

Clone o repositório:

Bash
git clone https://github.com/jaaodev/maratona-java.git
Entre na pasta do projeto:

Bash
cd maratona-java
Compile e execute a classe desejada:

Bash
javac javacore/src/Kpolimorfismo/test/ProdutoTest01.java
java java -cp javacore/src Kpolimorfismo.test.ProdutoTest01
